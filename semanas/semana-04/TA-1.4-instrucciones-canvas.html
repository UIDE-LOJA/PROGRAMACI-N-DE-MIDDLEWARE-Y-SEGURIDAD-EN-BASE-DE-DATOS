Ejemplo de ítem del checklist:

Categoría: Autenticación
Ítem: Implementar JWT con firma HMAC-SHA256

Justificación:
JWT permite autenticación stateless escalable. HMAC-SHA256 garantiza 
integridad del token y previene tampering. Tokens con expiración corta 
(15 min) reducen ventana de ataque si son comprometidos.

Amenaza mitigada: Spoofing, Tampering

Ejemplo de implementación (Node.js):
const jwt = require('jsonwebtoken');
const token = jwt.sign(
  { userId: user.id, role: user.role },
  process.env.JWT_SECRET,
  { expiresIn: '15m', algorithm: 'HS256' }
);

Criterio de verificación:
✓ Tokens firmados con HMAC-SHA256
✓ Expiración configurada (≤ 15 min)
✓ Secret almacenado en variable de entorno
✓ Validación de firma en cada request
