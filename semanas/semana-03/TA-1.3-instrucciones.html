Esta actividad de trabajo autónomo te permitirá crear una colección de Postman y un script de cliente automatizado para testing de APIs con diferentes protocolos. Te enfocarás en automatizar pruebas, validar respuestas, y documentar casos de uso de los protocolos estudiados en la Semana 3.

¿Qué realizarás?

Componente Instrucciones específicas Criterios de calidad
Información personal Incluye tus datos personales (nombre completo, curso, fecha) en el README del proyecto. Información completa y presentación profesional.
Colección Postman Crea una colección con mínimo 5 requests: GET, POST (JSON), POST (form-data), PUT, DELETE. Incluye tests automáticos que validen status code y estructura de respuesta. Colección funcional con tests automatizados y bien organizada.
Script de cliente Desarrolla un script (Node.js, Python o similar) que consuma una API pública (ej.: JSONPlaceholder, REST Countries) realizando al menos 3 operaciones diferentes. Maneja errores y muestra resultados en consola. Script funcional con manejo de errores y salida clara.
Comparación de protocolos Tabla comparativa entre HTTP/2, WebSockets y gRPC: casos de uso, ventajas, desventajas, cuándo usar cada uno. Comparación técnica precisa basada en material de clase.
Documentación README.md con: instrucciones de uso de la colección Postman, cómo ejecutar el script, explicación de las pruebas realizadas, y tabla comparativa de protocolos. Documentación clara, completa y bien estructurada.

Ejemplo de test en Postman:

pm.test("Status code es 200", function () {
    pm.response.to.have.status(200);
});

pm.test("Respuesta contiene campo 'data'", function () {
    var jsonData = pm.response.json();
    pm.expect(jsonData).to.have.property('data');
});

APIs públicas sugeridas para el script:

  • JSONPlaceholder: https://jsonplaceholder.typicode.com/
  • REST Countries: https://restcountries.com/

Elementos a incluir en la tabla comparativa:

  • Casos de uso: Cuándo es más apropiado cada uno
  • Ventajas: Principales beneficios
  • Desventajas: Limitaciones o complejidades
