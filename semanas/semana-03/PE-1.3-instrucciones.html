Esta actividad práctica te permitirá implementar un servicio de eco en Node.js que responda en JSON y Protocol Buffers, aplicando los conceptos de serialización y protocolos de comunicación vistos en la Semana 3. Te enfocarás en crear un servicio funcional, manejar múltiples formatos, y realizar testing de performance.

¿Qué realizarás?

Actividad Instrucciones Entregable
1) Configuración inicial Descarga el proyecto base PE-1.3. Instala dependencias con npm install Proyecto funcionando
2) Ejecutar servidor Inicia el servidor con npm start y verifica que corre en puerto 3000 Captura del servidor iniciado
3) Probar endpoints Prueba el endpoint JSON con Postman o curl. Observa el tamaño de la respuesta en los headers Captura de petición y respuesta JSON
4) Comparación automática Ejecuta npm test para comparar automáticamente JSON vs Protobuf (5 peticiones a cada endpoint) Captura de resultados de comparación
5) Análisis de resultados Completa la tabla de análisis en el README con tus observaciones sobre tamaños y diferencias README.md con análisis completado

Estructura mínima del proyecto:

PE-1.3/
├── package.json
├── server.js
├── echo.proto
├── test.js
├── capturas/
└── README.md

Ejemplo de esquema .proto

syntax = "proto3";

message EchoRequest {
  string message = 1;
}

message EchoResponse {
  string message = 1;
  int64 timestamp = 2;
}

Comandos principales:

npm install    # Instalar dependencias
npm start      # Iniciar servidor
npm test       # Ejecutar comparación

Ejemplo de prueba con curl:

curl -X POST http://localhost:3000/echo/json \
  -H "Content-Type: application/json" \
  -d '{"message": "Hola mundo"}'

Criterios de evaluación:

  • Servidor funciona correctamente (40%)
  • Pruebas realizadas y documentadas con capturas (30%)
  • Análisis de resultados completado en README (30%)
