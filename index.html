Compendios Semanales

Contenidos organizados cronológicamente por semanas

Semana 01
Fundamentos de Sistemas Distribuidos
Unidad 1: Conceptos de SD, roles del middleware, panorama de tecnologías, estilos y capas a alto nivel
Semana 02
Arquitectura de 3 Capas y MVC
Unidad 1: 3 capas, MVC, comparación de estilos, middleware como integración
Semana 03
Protocolos y Formatos de Datos
Unidad 2: HTTP/2, WebSockets, gRPC, formatos: JSON, Protobuf, MessagePack, XML básico y rol enterprise
Semana 04
Seguridad: CIA y Criptografía
Unidad 2: CIA, modelos de amenaza, criptografía aplicada (hash, sim/asim, TLS, PKI) orientada a APIs
Semana 05
Evaluación Diagnóstica
Evaluación: Evaluación de conocimientos adquiridos en las primeras 4 semanas
Semana 06
Desarrollo de APIs con Django / Node.js
Unidad 3: Django REST Framework en profundidad / TypeScript + Node.js, Express y Fastify (ecosistema), middlewares personalizados, validación (zod/Joi), estructura de proyecto
Semana 07
Diseño de APIs REST Escalables y Contratos MCP
Unidad 3: Patrones avanzados de API (HATEOAS, Pagination, Filtering), versiones y documentación automatizada (Swagger/OpenAPI), optimización y caching, estructura de endpoints orientados a herramientas MCP
Semana 08
Seguridad y Autenticación de APIs
Unidad 4: Autenticación y autorización, OAuth2/OIDC (passport/openid-client), tokens (JWT, PASETO), protección ante XSS/CSRF/SSRF, rate limiting y CORS
Semana 09
Optimización y Persistencia
Unidad 4: ORM/ODM (Prisma/TypeORM/Sequelize, Mongoose), N+1, caching con Redis, sharding/particionamiento, NoSQL como middleware de datos, observabilidad básica (logs/metrics)
Semana 10
Evaluación Formativa
Evaluación: Evaluación de conocimientos de desarrollo de APIs y seguridad
Semana 11
Seguridad en Base de Datos
Unidad 5: Modelos de seguridad, RBAC/ABAC, cifrado en reposo/en tránsito, auditoría y detección de intrusiones
Semana 12
Integración con Sistemas Legacy y Servidores MCP
Unidad 5: SOAP/WSDL, integración legacy, WS-Security, migración SOAP→REST/GraphQL. Protocolo MCP: estructura, registro de herramientas y capacidades, comunicación stdio/WebSocket
Semana 13
Arquitectura de Mensajerías
Unidad 6: RabbitMQ/Kafka en Node (amqplib/kafkajs), Pub/Sub, Saga, CQRS, EDA, resiliencia (reintentos, DLQ, idempotencia)
Semana 14
Microservicios y Service Mesh
Unidad 6: Diseño y descomposición de dominios, Service Mesh (Istio/Linkerd), estrategias de migración, observabilidad (tracing con OpenTelemetry)
Semana 15
Despliegue e Integración Continua
Unidad 7: Integración continua y policy as code (aprobación de herramientas), secrets management y rotación, despliegue Canary/Blue-Green para nuevas herramientas MCP, auditoría de despliegues
Semana 16
Evaluación Sumativa
Evaluación: Evaluación final del curso
